스마트공장 재고·물류관리 시스템 v9 (납품형)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 실행 방법
  1. index.html 더블클릭
  2. 로그인 화면에서 아래 계정으로 접속

▶ 기본 로그인 계정 (납품 후 변경 권장)
  ┌──────────┬──────────────┬──────────────┐
  │ 아이디   │ 비밀번호     │ 권한         │
  ├──────────┼──────────────┼──────────────┤
  │ admin    │ admin1234    │ 관리자 (전체)│
  │ staff1   │ 1234         │ 직원         │
  │ view1    │ 1234         │ 조회전용     │
  └──────────┴──────────────┴──────────────┘

▶ 고객사 커스터마이징 (index.html 상단 CONFIG 블록)
  CLIENT_ID   : 영문 소문자 (localStorage 키 구분용)
  COMPANY     : 화면에 표시될 회사명
  LOGO_TEXT   : 로고 2~3자 (예: KY, ABC)
  THEME_C1/C2 : 기본 컬러 (설정 탭에서도 변경 가능)
  GAS_URL     : Google Apps Script URL (빈값=로컬 전용)
  WAREHOUSES  : 창고명 배열 (예: ['본사창고','지점창고'])
  USERS       : 계정 목록 (id/pw/name/role)
  SEED_ITEMS  : 초기 샘플 품목

▶ 관리자 설정 탭에서 변경 가능한 항목
  - 회사명 / 로고 문자
  - 메인 컬러 테마 (6가지 프리셋)
  - 기본 담당자 / 기본 창고
  - Google Sheets 연동 URL
  - 페이지당 표시 건수

▶ 주요 기능
  - 입출고 등록 (바코드 스캔 지원)
  - 입출고 이력 11컬럼 조회 (누적재고 포함)
  - 다중 필터 + 빠른 기간 버튼
  - 엑셀(xlsx) / CSV 다운로드
  - 부족재고 알림 + 배지
  - Google Sheets 자동 동기화
  - 세션 자동복구 (8시간)

▶ 데이터 저장
  - 브라우저 localStorage 저장 (오프라인 동작)
  - GAS URL 설정 시 Google Sheets 자동 연동
  - Chrome / Edge 권장

▶ 주의사항
  - 브라우저 방문기록 삭제 시 로컬 데이터 초기화됨
  - 중요 데이터는 Google Sheets 연동 사용 권장
