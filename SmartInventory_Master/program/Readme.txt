KY Smart Inventory Pro v10
제조업 맞춤 재고·물류관리 시스템

[프로그램 개요]
본 프로그램은 중소 제조업체, 유통업체를 위한
실시간 재고관리 및 입출고 관리 시스템입니다.

바코드 입력, 구글폼 연동, 구글시트 동기화,
실시간 재고조회, 품목관리 기능을 제공합니다.

[주요 기능]

1. 품목마스터 관리

* 바코드 등록
* 품목명 관리
* 재고 기준 관리

2. 입출고 관리

* 수기 입력
* 바코드 입력
* 구글폼 자동 입력 연동

3. 실시간 재고조회

* 현재고 자동 계산
* 부족재고 경고

4. 관리자 설정

* 회사명 변경
* 로고 문자 / 이미지 변경
* 색상 테마 변경
* GAS URL 설정

5. 데이터 백엔드

* Google Sheets
* Google Apps Script

[실행 방법]

1. index.html 더블클릭
2. 로그인
3. 설정창에서 GAS URL 입력
4. 저장 후 사용

[권장 브라우저]
Google Chrome / Microsoft Edge 최신버전

[관리자 계정]
최초 계정은 납품 시 별도 설정

[주의사항]

* 인터넷 연결 상태 권장
* 구글시트 삭제 금지
* Apps Script URL 변경 시 설정 재입력 필요
